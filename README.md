        กระบวนการที่ 6 การรับแจ้งชำระเงิน ของพนักงาน  
        กระบวนการที่ 7 การตรวจสอบการทำสัญญาเช่า ของพนักงาน 
        กระบวนการที่ 8 การจัดทำรายงาน ของพนักงาน และ ผู้ดูและระบบ
